ℹ️ 使用说明

系统使用指南与注意事项

🚀 快速开始
# 步骤1：安装依赖
pip install flask opencv-python pillow scikit-learn numpy pandas scipy

# 步骤2：启动后端（保持终端运行）
cd src/system/backend
python image_app.py

# 步骤3：浏览器打开前端
双击 src/system/frontend/grape_system.html
📸 拍照建议
✅ 正面拍摄，葡萄占画面60%以上
✅ 自然光线，避免强烈反光
✅ 清晰对焦，背景简洁
✅ 单串或单粒葡萄特写
❌ 避免夜晚强光闪照
❌ 避免图片模糊或遮挡
⚠️ 重要说明
• 图像模型基于颜色特征识别
• 实际准确率受光照和品种影响
• 高光谱模型准确率更高（87.7%）
• 图像识别作为快速辅助判断
• 建议结合人工判断综合评估
